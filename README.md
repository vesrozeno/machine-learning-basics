# machine learning basics

During the [SuperDataScience](https://www.superdatascience.com/) [Machine Learning A-Z Udemy course](https://www.udemy.com/course/machinelearning/), I had the opportunity to implement various machine learning algorithms and techniques. It was an excellent course that provided me with a valuable overview of what machine learning entails.
Throughout the course, I utilized Python in [Google Colab](https://colab.google/) for coding purposes.
